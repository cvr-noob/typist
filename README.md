## Setup

> **Important:** This project currently works only with Python on Windows.

### 1. Create a virtual environment

#### Windows

```bash
python -m venv .venv
.\.venv\Scripts\activate
```

#### Unix-based OS

Not supported at the moment.

---

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 3. Add the text to type

Paste the content you want to auto-type into the `code.txt` file.

---

### 4. Run the script

```bash
python main.py
```

After running the command, quickly place your cursor inside the target editor within **2 seconds**.
