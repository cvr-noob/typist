### Setup

#### IMP: Works only for Python on Windows

  1. Create virtual environment
    a. Windows
      ```bash
      python -m venv .venv
      .\.venv\Scripts\activate
      ```
    
    b. Unix-based OS
      It will not work on Unix-based OS
  
  2. Install dependencies
    ```bash
    pip install -r requirements.txt
    ```
  
  3. Paste the code to type in `code.txt` file.
  
  4. Run `main.py` and place the cursor on the editor within 2 seconds
  ```
  python main.py
  ```
